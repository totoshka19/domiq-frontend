Создай git коммит по спецификации Conventional Commits 1.0.0.

## КРИТИЧЕСКИ ВАЖНО
- НИКОГДА не добавлять `Co-Authored-By` — ни в тело, ни в футер
- Сообщение коммита ТОЛЬКО на русском языке (тип и область — английские)
- Коммит создаётся от имени разработчика, без упоминания AI

## Шаги

1. Запусти `git status` и `git diff --staged`.
   Если ничего не проиндексировано — запусти `git diff` и спроси какие файлы добавить.

2. Проанализируй изменения и определи тип, область, описание.

3. Типы:
   | Тип | Когда |
   |-----|-------|
   | `feat` | Новый компонент, страница, функция |
   | `fix` | Исправление бага |
   | `refactor` | Рефакторинг без изменения поведения |
   | `style` | Стили, CSS, Tailwind-классы |
   | `test` | Тесты |
   | `docs` | Документация |
   | `chore` | Зависимости, конфиги |
   | `perf` | Производительность, lazy loading |
   | `build` | Vite конфиг, сборка |
   | `ci` | CI/CD, Vercel конфиг |

4. Области (scope) для фронтенда:
   `auth` · `listings` · `search` · `chat` · `profile` · `admin` · `ui` · `api` · `store` · `router` · `deps` · `maps`

5. Правила:
   - Повелительное наклонение: "добавить", "исправить"
   - Строчные буквы, без точки в конце
   - Первая строка ≤ 72 символов

6. Покажи предлагаемое сообщение и спроси подтверждение:
   ```
   Предлагаемый коммит:

   feat(listings): добавить компонент карточки объявления

   Включает галерею фото, цену, адрес и кнопку избранного.
   Адаптивен от 375px до 1920px.

   Закоммитить? (да / изменить / отмена)
   ```

7. При подтверждении — только `-m`, никаких `--trailer`:
   ```bash
   git add <файлы>
   git commit -m "feat(listings): добавить компонент карточки объявления" \
              -m "Включает галерею фото, цену, адрес и кнопку избранного."
   ```

8. Покажи `git log --oneline -5`.

## Примеры коммитов для Domiq Frontend
```
feat(listings): добавить карточку объявления с галереей
feat(search): добавить панель фильтров с ценой и комнатами
feat(maps): интегрировать Яндекс.Карты на страницу выдачи
fix(auth): исправить редирект после логина
fix(listings): исправить бесконечную загрузку при пагинации
style(ui): обновить стили кнопок по дизайну
refactor(api): вынести типы запросов в src/types/
perf(listings): добавить lazy loading для изображений
feat(chat): реализовать WebSocket-чат в реальном времени
chore(deps): обновить TanStack Query до v5
```
